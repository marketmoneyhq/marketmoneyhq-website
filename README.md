# Market Money HQ

Premium education and mentorship website for Market Money HQ — helping people build financial independence through trading education, business growth, and AI & digital skills.

## Stack

- Next.js 15 (App Router)
- React 19 + TypeScript
- Tailwind CSS 4
- Framer Motion
- next-themes (dark mode)

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm start` | Start production server |
| `npm run lint` | Run ESLint |

## Pages

- `/` — Home
- `/about` — About & mission
- `/services` — All services
- `/trading` — Trading education
- `/website-design` — Website & marketing
- `/ai` — AI consulting
- `/resources` — Educational resources
- `/contact` — Contact & consultation
